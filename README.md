# vim-denite-tab

`vim-denite-tab` is a source for denite.nvim

# Requirements

- [Shougo/denite.nvim](https://github.com/Shougo/denite.nvim)

# Install

## dein.vim

```
call dein#add('w1mvy/vim-denite-tab')
```

# Usage

List your tabpages

```
:Denite tab
```
